Batch Update Automation Rules

securityhub_batch_update_automation_rules R Documentation

Updates one or more automation rules based on rule Amazon Resource Names (ARNs) and input parameters¶

Description¶

Updates one or more automation rules based on rule Amazon Resource Names (ARNs) and input parameters.

Usage¶

securityhub_batch_update_automation_rules(
  UpdateAutomationRulesRequestItems)

Arguments¶

UpdateAutomationRulesRequestItems

[required] An array of ARNs for the rules that are to be updated. Optionally, you can also include RuleStatus and RuleOrder.

Value¶

A list with the following syntax: 

list(
  ProcessedAutomationRules = list(
    "string"

  UnprocessedAutomationRules = list(
    list(
      RuleArn = "string",
      ErrorCode = 123,
      ErrorMessage = "string"
    )
  )
)
